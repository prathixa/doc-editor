Developed a "Google Docs" style editor that allows multiple users to collaborate on the same document simultaneously with zero lag.

Tech Stack Used:
- Frontend: React.js
- Editor Engine: Quill.js
- Real-time Sync: Socket.io
- Backend: Node.js & Express
- Database: MongoDB 

## Key Features:
- Live Collaboration: Changes reflect instantly across all sessions.
- Persistence: Documents are saved and retrieved from MongoDB.
- Clean UI: Responsive design with a professional document layout.
